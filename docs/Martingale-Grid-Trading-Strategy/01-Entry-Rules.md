> 适用阶段：空仓观察、等待首仓、首仓建立中

## 一、价格校准规则

### 1.1 概念定义

| 概念 | 定义 |
|:--|:--|
| 三级支撑位 | 近三个月价格触及后反弹的三个成交密集区，按距当前价格由近到远排列 |
| 三级压力位 | 近三个月价格触及后回落的三个成交密集区，按距当前价格由近到远排列 |
| 理想首仓价格区间 | 一级支撑位至一级支撑位上方约 1.5%~2% 的范围 |
| 可接受首仓最高价 | 理想首仓价格区间上沿上方约 1%~2% |
| 禁止追高价格 | 可接受首仓最高价之上，风险收益比已跌破 1.8。做空时方向相反 |
| 风险收益比（做多） | (一级压力位 - 当前价格) / (当前价格 - 一级支撑位) |
| 风险收益比（做空） | (当前价格 - 一级支撑位) / (一级压力位 - 当前价格) |

### 1.2 做多首仓：价格位置与操作对应关系

```
支撑区 ←──理想首仓──→ 可接受区 ←──观望区──→ 压力区
  ↑          ↑           ↑        ↑         ↑
一级支撑    理想上沿     可接受上沿   禁止追高    一级压力

理想首仓区间：立即建仓，建议首仓占比 0.20 ~ 0.30
可接受区间：择机建仓，建议首仓占比 0.10 ~ 0.20
观望区间（距支撑 > 3% 或距压力 < 3%）：禁止建仓，等待回落
```

### 1.3 做多禁止条件

以下任一条触发，禁止大幅首仓：

1. 距一级支撑位 > 3%
2. 距一级压力位 < 3%
3. 处于近 5 日区间上半部
4. 刚经历急涨（近 2 日涨幅 > 5% 且无回调）
5. 近 24 小时触及区间上沿
6. 风险收益比 < 1.8
7. 未来一周存在重大不确定事件
8. 预测置信度 < 0.65
9. 成交量在接近压力位时明显萎缩（上攻无力）

### 1.4 做空首仓：价格位置与操作对应关系

```
压力区 ←──理想首仓──→ 可接受区 ←──观望区──→ 支撑区
  ↑          ↑           ↑        ↑         ↑
一级压力    理想下沿     可接受下沿   禁止追空    一级支撑

理想首仓区间：立即建空仓，建议首仓占比 0.15 ~ 0.25
可接受区间：择机建空仓，建议首仓占比 0.08 ~ 0.15
观望区间：禁止建空仓，等待反弹
```

### 1.5 做空禁止条件

1. 距一级压力位 > 3%
2. 距一级支撑位 < 3%
3. 处于近 5 日区间下半部
4. 大盘牛市或股票基本面优良（做空大型牛股是自杀行为）
5. 风险收益比 < 1.8
6. 预测置信度 < 0.65

### 1.6 通用规则

所有价格数据必须逻辑自洽：当前价格、近 24 小时区间、近三个月区间、三级支撑位、三级压力位——任意两者之间的相对关系必须合理。

## 二、首仓择时规则

### 2.1 首仓建议的三种取值

| 取值 | 含义 | 适用条件 |
|:--|:--|:--|
| 禁止 | 当前价格不适合任何首仓操作。不建仓，不试探。 | 价格不在理想/可接受区间，或触发任一禁止条件 |
| 择机 | 价格已在可接受范围内，等待 CCI 等小周期信号确认后入场。 | 价格在可接受区间但非理想区间，或理想区间但缺少小周期确认 |
| 立即 | 价格已处于低风险首仓区，不等待信号，立即建仓。 | 价格位于理想首仓区间，CCI 方向一致，成交量配合，无事件风险，置信度 ≥ 0.75 |

### 2.2 做多首仓决策框架

须逐一检查以下条件，全部满足方可输出「立即」，至少满足大多数方可输出「择机」：

| 检查项 | 通过标准 | 不通过时处理 |
|:--|:--|:--|
| 1. 距一级支撑位距离 | ≤ 3% | 禁止 |
| 2. 距一级压力位距离 | 首选：≥ 5%；次选：> 支撑距离 × 2 | 首选不通过→择机；两项均不通过→禁止 |
| 3. 近 5 日价格分位 | < 0.5 | 禁止 |
| 4. 近 2 日走势 | 无急涨（< 5%），或急涨后已有明显回调 | 择机或禁止 |
| 5. 风险收益比 | ≥ 1.8 | 禁止 |
| 6. 预测置信度 | ≥ 0.65 | 禁止 |
| 7. 未来一周事件风险 | 无重大不确定事件 | 降级处理 |
| 8. 量价关系 | 支撑位附近成交量未异常放大 | 禁止 |
| 9. CCI/RSI 小周期信号 | CCI 在 -100 附近或出现底部背离 | 择机等待确认 |
| 10. 同板块走势 | 板块未出现集体性大幅下挫 | 降级处理 |

### 2.2b 做空首仓决策框架

须逐一检查以下条件，全部满足方可输出「立即」，至少满足大多数方可输出「择机」：

| 检查项 | 通过标准 | 不通过时处理 |
|:--|:--|:--|
| 1. 距一级压力位距离 | ≤ 3% | 禁止 |
| 2. 距一级支撑位距离 | 首选：≥ 5%；次选：> 压力距离 × 2 | 首选不通过→择机；两项均不通过→禁止 |
| 3. 近 5 日价格分位 | > 0.5 | 禁止 |
| 4. 近 2 日走势 | 无急跌（< 5%），或急跌后已有明显反弹 | 择机或禁止 |
| 5. 风险收益比 | ≥ 1.8 | 禁止 |
| 6. 预测置信度 | ≥ 0.65 | 禁止 |
| 7. 未来一周事件风险 | 无重大不确定事件 | 降级处理 |
| 8. 大盘环境 | 大盘偏弱或中性（牛市禁止做空） | 禁止 |
| 9. CCI/RSI 小周期信号 | CCI 在 +100 附近或出现顶部背离 | 择机等待确认 |
| 10. 同板块走势 | 板块未出现集体性大幅拉升 | 降级处理 |

#### 检查项 #2 的层次判定逻辑

「距关键位距离」采用首选/次选两级判定，避免歧义：

1. **首选**：距反方向一级关键位 ≥ 5% → 直接通过，价格有充分运行空间
2. **次选**：不满足首选，但距反方向关键位 > 同向关键位距离的 2 倍 → 降级为**择机**（空间勉强达标，需等待更多确认信号）
3. **两项均不通过** → **禁止**建仓（风险收益比不足）

**示例（做多）：** 一级支撑 S₁ = $100，当前价 $102，一级压力 P₁ = $106
- 支撑距离 = $2 → 2%，压力距离 = $4 → 3.8%
- 首选：3.8% < 5% → ✗
- 次选：3.8% > 2% × 2 = 4%？→ 3.8% < 4%，✗
- 结论：禁止建仓，压力距离不足以支撑网格运行

**示例（做空）：** 一级压力 R₁ = $200，当前价 $198，一级支撑 S₁ = $185
- 压力距离 = $2 → 1%，支撑距离 = $13 → 6.5%
- 首选：6.5% ≥ 5% → ✓
- 结论：直接通过


### 2.3 思维陷阱——反复警惕的错误推理

| 陷阱 | 正确认知 |
|:--|:--|
| 「这股长期看涨，现在买没问题」 | 看涨 ≠ 当前价格适合首仓。差价越大，网格效率越低。 |
| 「这股长期看跌，现在空没问题」 | 看跌 ≠ 当前价格适合首空。差价越大，网格效率越低。 |
| 「好公司、好赛道，不怕拿」 | 好公司 ≠ 任何时候都是好买点。好公司买贵了，网格照样摊不平。 |
| 「未来一周可能涨，先建仓占位置」 | 方向预测 ≠ 入场时点选择。先在高位建仓，再等回调，已经伤了策略根基。 |
| 「未来一周可能跌，先建空仓占位置」 | 方向预测 ≠ 入场时点选择。先在低位建空仓，再等反弹，已经伤了策略根基。 |
| 「技术指标看着还行，买一点试试」 | 指标信号 ≠ 价格安全。少量试探仓也会随着不利波动触发加仓，形成真实风险。 |
| 「技术指标看着还行，空一点试试」 | 指标信号 ≠ 价格安全。做空试探仓随着不利上涨触发加空仓，同样形成真实风险。 |

## 三、信号强度 → 仓位映射

### 3.1 仓位核心公式

```
建议首仓占比 + 网格资金占比 ≤ 最大仓位占比
```

- 首仓不可直接打满最大仓位占比
- 最大仓位占比高 ≠ 现在应该满仓——它是上限，不是目标
- 首仓位置不理想时，优先降建议首仓占比，而非放宽价格标准

### 3.2 做多仓位映射

| 信号组合 | 建议首仓占比 | 最大仓位占比 |
|:--|:--|:--|
| 价格在理想首仓区 + CCI 底部确认 + 量能放大 + 板块共振 + 无事件风险 | 0.25 ~ 0.35 | 0.80 ~ 0.90 |
| 价格在理想首仓区 + CCI 底部确认 + 量能配合（少 1-2 项） | 0.20 ~ 0.30 | 0.70 ~ 0.80 |
| 价格在可接受区间 + CCI 偏弱 + 量能一般 | 0.10 ~ 0.20 | 0.55 ~ 0.70 |
| 价格在可接受区间但 CCI 未确认 + 存在小事件风险 | 0.05 ~ 0.10 | 0.30 ~ 0.55 |
| 价格不在区间内，或触发任何禁止条件 | 0 | 0.10 ~ 0.50 |
| 标的质量差、支撑压力模糊、波动率不适合网格 | 0 | 0 |

### 3.3 做空仓位映射

做空仓位应比做多更保守：

| 信号组合 | 建议首仓占比 | 最大仓位占比 |
|:--|:--|:--|
| 价格在理想首空区 + CCI 顶背离 + 板块共振 + 大盘偏弱 | 0.15 ~ 0.25 | 0.50 ~ 0.65 |
| 价格在可接受区间 + CCI 超买 + 大盘中性 | 0.08 ~ 0.15 | 0.30 ~ 0.50 |
| 条件部分缺失，或大盘偏牛 | 0.03 ~ 0.08 | 0.15 ~ 0.30 |
| 任何禁止条件触发 | 0 | 0 |

## 四、最小交易单位容差规则

股票、ETF 等品种存在最小交易单位，可能导致实际建仓金额无法精确等于目标金额。用于判断仓位是否「有效建立」，避免策略卡在「首仓建立中」无法流转。

### 4.1 核心公式

```
每手金额 = 当前价格 × 每手股数
实际可买手数 = floor(建议首仓金额 / 每手金额)  或  ceil(建议首仓金额 / 每手金额)
实际首仓金额 = 实际可买手数 × 每手金额
实际首仓占比 = 实际首仓金额 / 总资金
```

### 4.2 容差判断三层规则（按优先级）

| 优先级 | 规则 | 条件 | 结果 |
|:--:|:--|:--|:--|
| 1 | **价格优先原则** | 价格已回升至非理想区，且实际仓位 ≥ 建议首仓占比 × 0.90 | 视为有效建立，流转至「网格管理期」，不得追高补仓 |
| 2 | **一手容差原则** | 实际首仓占比 ≥ 建议首仓占比 - 每手金额/总资金（差额不足一手） | 视为有效建立，流转至「网格管理期」 |
| 3 | **深度不足原则** | 价格仍在有利区间，但差额 ≥ 一手金额，且实际仓位 < 建议首仓占比 × 0.85 | 停留在「首仓建立中」，等待有利价格补仓 |

### 4.3 示例

```
总资金 = 100,000，建议首仓占比 = 0.25 → 目标金额 = 25,000
股价 = 123.45，每手金额 = 12,345

可选方案：
  200股 = 24,690（占比 24.69%，差额 310）
  300股 = 37,035（占比 37.04%，超额 12,035）

规则2判定：24.69% ≥ 25% - 12.345% = 12.655% ✓ → 差额不足一手
→ 视为有效建立，进入「网格管理期」
```

```
总资金 = 100,000，建议首仓占比 = 0.25 → 目标金额 = 25,000
股价 = 450.00，每手金额 = 45,000

可选方案：0股 或 100股 = 45,000（占比 45%）

规则3判定：0% < 25% × 0.85 = 21.25% → 差额远超一手
→ 若价格在有利区间，继续等待建仓机会
→ 若价格已回升至非理想区，规则1判定：0% < 25% × 0.90 = 22.5% → 无法有效建立，退回「等待首仓」
```
