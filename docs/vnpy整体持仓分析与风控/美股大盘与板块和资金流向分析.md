# 美股市场分析框架（AI 智能体作业指导书）

## 一、分析目标

回答四个核心问题：

```text
1. 大盘方向？   ↑ 上涨  ↓ 下跌  → 震荡
2. 当前板块主线是什么？  AI / 半导体 / 金融 / 能源 / 其他
3. 当前市场风格是什么？  大盘/小盘  成长/价值
4. 风险在哪里？  利率 / 流动性 / 财报 / 地缘政治 /  其他
```

---

## 二、分析层次

```text
宏观层（决定方向）  →  权重最高
    ↓
行业层（决定板块）
    ↓
风格层（决定资金偏好）
    ↓
情绪与结构层（决定风险）
```

---

## 三、执行流程（AI 请按此顺序执行）

### Step 0：确定数据获取方式

本模块所有指标，AI 应优先使用以下外部技能获取数据：

| 数据类别 | 推荐获取方式 | 说明 |
|---------|------------|------|
| 美联储消息 / FOMC / 鲍威尔讲话 | `web_search` | 搜索最新 FOMC 声明、点阵图、讲话摘要 |
| US10Y / 2Y / 30Y 收益率 | `web_search` 或 `yahoo-finance` | 搜索 "US10Y yield today" |
| DXY 美元指数 | `web_search` | 搜索 "DXY index today" |
| VIX 恐慌指数 | `web_search` | 搜索 "VIX index today" |
| SPY / QQQ / IWM 价格 | `yahoo-finance` | 获取实时价格和涨跌幅 |
| 板块 ETF（XLF/XLE/XLV 等） | `yahoo-finance` | 获取板块表现数据 |
| 龙头个股（NVDA/AMD 等） | `yahoo-finance` | 获取价格和技术指标 |
| 宏观经济数据（CPI/非农/PMI） | `web_search` | 搜索最新发布值 |
| 涨跌家数 / 新高新低 / A/D Line | `web_search` | 搜索 "NYSE advance decline today" |
| Put/Call Ratio | `web_search` | 搜索 "put call ratio today" |
| CNN Fear & Greed | `web_search` | 搜索 "fear and greed index today" |

### Step 1：宏观层分析

按顺序获取并分析以下数据：

1. **美联储立场**
   - 获取方式：`web_search` 搜索 "latest FOMC statement" 或 "Fed latest news"
   - 判断：鸽派 → 利多 / 鹰派 → 利空

2. **美债收益率（US10Y）**
   - 获取方式：`web_search` 搜索 "US10Y yield today"
   - 判断：10Y > 4.5% 纳指承压，> 5.0% 显著压制
   - 趋势判断：收益率上行 → 利空科技股 / 下行 → 利好成长股

3. **美元指数（DXY）**
   - 获取方式：`web_search` 搜索 "DXY index today"
   - 判断：DXY 上涨 → 风险资产承压 / 下跌 → 风险资产偏好提升

4. **流动性**
   - 获取方式：`web_search` 搜索 "Fed balance sheet" "RRP facility"
   - 核心问题：市场有没有增量资金

5. **宏观经济数据**
   - 获取方式：`web_search` 搜索最新 CPI / PPI / 非农 / PMI / GDP
   - 最佳组合：经济强 + 通胀下降 = Soft Landing

**宏观结论输出**：大盘方向（看多 / 看空 / 震荡）+ 关键依据

### Step 2：板块轮动分析

获取各板块核心标的的最新表现：

| 板块 | 观察标的 | 获取方式 | 判断 |
|------|---------|---------|------|
| AI 链 | NVDA, MSFT, META, GOOG, AMZN | `yahoo-finance` | NVDA 强 → AI 强 → 风险偏好高 |
| 半导体 | SOXX, AMD, AVGO, TSM, MU | `yahoo-finance` | 科技牛市发动机 |
| 金融 | XLF, JPM, GS, BAC, MS | `yahoo-finance` | 经济健康程度 |
| 能源 | XLE, 原油 | `yahoo-finance` / `web_search` | 通胀预期 |
| 防御板块 | XLV(医疗), XLP(消费), XLU(公用) | `yahoo-finance` | 避险情绪 |

**板块轮动口诀**（用于判断当前处于哪个阶段）：

```text
AI领涨 → 半导体跟涨 → 软件跟涨 → 小盘股补涨 → 周期股补涨 → 防御板块接棒 → 熊市
```

**板块结论输出**：当前主线板块 + 轮动阶段 + 依据

### Step 3：风格轮动分析

| 观察对象 | 获取方式 | 判断 |
|---------|---------|------|
| SPY, QQQ, IWM | `yahoo-finance` | QQQ/SPY 比值判断成长风格 |
| VUG, VTV | `yahoo-finance` | VUG 强 = 牛市风格 / VTV 强 = 防御风格 |

**风格结论输出**：大盘/小盘 + 成长/价值

### Step 4：市场情绪与风险分析

| 指标 | 获取方式 | 阈值判断 |
|------|---------|---------|
| VIX | `web_search` "VIX today" | < 15 乐观 / 15-25 正常 / > 30 恐慌 |
| Put/Call Ratio | `web_search` "put call ratio" | 高=看空 / 低=看多 |
| Fear & Greed | `web_search` "fear greed index" | 辅助参考，不单独使用 |
| 涨跌家数 / A/D Line | `web_search` | 指数涨但 A/D 降 = 假突破 |

**风险结论输出**：主要风险点 + 风险等级

### Step 5：汇总输出

按下方输出模板生成结构化 JSON 结果。

---

## 四、风险等级与建议行动

| 风险等级 | 触发条件 | 建议行动 |
|---------|---------|---------|
| **低风险** | VIX < 15, 宏观稳定, 板块健康 | 维持现有策略，仅记录观察 |
| **中等风险** | VIX 15-25, 10Y > 4.5%, 板块轮动到防御 | 提示关注，不建议主动干预 |
| **高风险** | VIX > 30, 10Y > 5%, DXY 急涨, A/D 持续背离 | 建议减仓，发送 notice 提醒策略 |
| **极高风险** | 黑天鹅事件, Fed 紧急加息, 流动性危机 | 建议大幅减仓或清仓，发送控制命令 |

**重要原则**：
- 低风险时：**不要干涉策略运行**，仅记录分析结果
- 中高风险时：先发 notice 通知，再判断是否需要调仓
- 极高风险时：可以发送 `set_target_pos 0` 命令减仓
