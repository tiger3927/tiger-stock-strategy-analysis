# 美股市场分析框架（AI 智能体作业指导书）

> **文档定位**：本文档是 `tiger-stock-strategy-analysis` 技能的子模块。当用户请求美股大盘分析、板块分析、资金流向分析时，本技能会加载此文档作为 AI 的执行指令。AI 应严格按本文档的【二、分析周期】确定周期后，再按【四、执行流程】逐步获取数据并输出结果。

## 一、分析目标

关于美股大盘与板块和资金流向的分析，回答四个核心问题：

```text
1. 大盘方向？   ↑ 上涨  ↓ 下跌  → 震荡
2. 当前板块主线是什么？  AI / 半导体 / 金融 / 能源 / 其他
3. 当前市场风格是什么？  大盘/小盘  成长/价值
4. 风险在哪里？  利率 / 流动性 / 财报 / 地缘政治 /  其他
```

## 二、分析周期（必须先确定）

AI 在执行分析前，**必须先确定分析周期**。不同周期关注不同指标，禁止拿月度数据解释盘中波动，也禁止拿日内数据判断中期趋势。

| 周期 | 时间跨度 | 核心关注指标 | 典型场景 |
|------|---------|-------------|---------|
| **盘中周期** | 当天 ~ 1 个交易日 | SPY/QQQ/IWM 实时涨跌、VIX 日内变化、US10Y 当日变动、DXY 当日变动、板块 ETF 涨跌、涨跌家数 | 日内决策、盘中风控、突发消息应对 |
| **短线周期** | 1 ~ 10 个交易日 | Fed 预期变化、10Y 趋势、DXY 趋势、板块轮动方向、市场广度（A/D Line）、龙头股表现 | 短期调仓、事件驱动、财报季 |
| **中期周期** | 1 ~ 3 个月 | CPI/PPI/非农/PMI/GDP、盈利预期、Margin Debt 趋势、IPO 募资规模、增发融资总额、估值水平 | 策略配置、仓位调整、趋势判断 |

### 周期与指标的对应关系

```text
盘中周期（实时/日度）          短线周期（周度）           中期周期（月度）
─────────────────────      ──────────────────       ──────────────────
SPY / QQQ / IWM 价格  ──→  SPY / QQQ / IWM 趋势 ──→  指数中期趋势
VIX 日内变化          ──→  VIX 均值/趋势        ──→  VIX 中枢
US10Y 当日变动        ──→  US10Y 趋势方向       ──→  US10Y 月度均值
DXY 当日变动          ──→  DXY 趋势方向          ──→  DXY 月度均值
板块 ETF 涨跌         ──→  板块轮动方向          ──→  板块相对强弱
涨跌家数              ──→  A/D Line 趋势         ──→  市场广度月度变化
                      ──→  龙头股表现            ──→  盈利预期趋势
                      ──→  Fed 预期变化          ──→  CPI / 非农 / GDP
                                                   ──→  Margin Debt 趋势
                                                   ──→  IPO / 增发融资规模
                                                   ──→  估值水平(PE/PB)
```

### AI 执行规则

1. **必须先确认周期**：如果用户没有明确指定，根据上下文推断（例如用户说「今天大盘怎么样」→ 盘中周期），默认推断未来一周的周期。
2. **优先获取对应周期的指标**：不要浪费 token 获取无关周期的数据
3. **不同周期的结论不可混淆**：盘中看多不代表中期看多，反之亦然

### Step 0.5：通用判断原则 —— 数值 + 趋势 + 市场确认

**单个指标本身不构成交易信号，必须经过三层验证：**

```text
┌─────────────────────────────────────────────────────┐
│  第一层：当前数值        →  是否处于历史极端区间？      │
│       ↓                                              │
│  第二层：近期趋势        →  正在往哪个方向加速？        │
│       ↓                                              │
│  第三层：市场是否确认？   →  相关资产是否已经在响应？    │
└─────────────────────────────────────────────────────┘
```

**核心逻辑**：指标是「因」，但只有市场承认的「因」才会变成「果」。如果市场不响应，指标再极端也只是噪音。

**错误示例 ❌**：
- 「US10Y > 4.5%，所以纳指承压，看空」—— 如果 QQQ 还在涨，这个判断是错的
- 「VIX > 25，市场恐慌」—— 如果 SPY 窄幅震荡，VIX 可能只是期权到期前的波动

**正确示例 ✅**：
- 「US10Y 突破 4.5% 且过去 5 日快速上行 + QQQ 连续 2 日下跌 + IWM 跑输 SPY → 利率压力有效，需要关注」
- 「US10Y 上行但 QQQ 继续创新高 + NVDA 领涨 → 市场暂时忽略利率，风险偏好仍强，不做空」

**所有 Step 中的阈值，都必须配合趋势和市场确认使用，禁止孤立判断。**

**🚫 数据缺失原则**：任何关键数据无法获取时，必须标记为 `missing`，**严禁猜测或编造**。数据缺失本身也是一种信息——它意味着分析结论的置信度需要下调。

### Step 1：宏观层分析

按顺序获取并分析以下数据：

1. **美联储立场**
   - 获取方式：`web_search` 搜索 "latest FOMC statement" 或 "Fed latest news"
   - 判断：鸽派 → 利多 / 鹰派 → 利空

2. **美债收益率（US10Y）** —— 三层判断
   - 获取方式：`web_search` 搜索 "US10Y yield today"
   - **数值**：> 4.5% 为压力区参考线，> 5.0% 为高压力区参考线
   - **趋势**：过去 5 日 / 20 日是否快速上行？缓慢上行？横盘？下行？
   - **市场确认**（必须检查）：
     - US10Y 上行 → 看 QQQ 是否下跌、IWM 是否跑输 SPY、成长股（NVDA/MSFT）是否承压
     - 若 QQQ 和 AI 龙头继续新高 → **市场忽略利率，不做空**
     - 若 QQQ 同步下跌 + 防御板块走强 → **利率压力确认，需要警惕**

3. **美元指数（DXY）** —— 三层判断
   - 获取方式：`web_search` 搜索 "DXY index today"
   - **数值**：> 105 为强美元参考线
   - **趋势**：过去 5 日 / 20 日是走强还是走弱？
   - **市场确认**：DXY 走强 + SPY/QQQ 下跌 + 新兴市场 ETF(EEM) 承压 → 美元压制确认
   - 若 DXY 走强但美股继续涨 → 美元压制无效，美股内生动力强

4. **流动性**
   - 获取方式：`web_search` 搜索 "Fed balance sheet" "RRP facility"
   - 核心问题：市场有没有增量资金

5. **宏观经济数据与利率预期解读**（核心）

   经济数据本身不直接驱动市场，**市场通过利率预期的棱镜解读数据**。同样一份数据，不同宏观背景下可能产生完全相反的市场反应。

   #### 5.1 预期差框架 —— 市场交易的是「意外」

   | 结果 | 定义 | 市场影响 |
   |------|------|---------|
   | 超预期 | 实际值 > 一致预期（如 CPI 3.5% vs 预期 3.3%） | 短期剧烈反应，方向取决于数据的「好坏」含义 |
   | 符合预期 | 实际值 ≈ 一致预期 | 通常已定价，反应平淡 |
   | 不及预期 | 实际值 < 一致预期 | 与超预期反向 |

   **获取方式**：`web_search` 搜索 "{指标名称} actual vs consensus" 或 "{指标名称} expectations"

   #### 5.2 利率预期传导链

   ```text
   经济数据公布
        ↓
   CME FedWatch 利率概率瞬间变化（加息/降息概率重新定价）
        ↓
   US10Y / US2Y 收益率跳动
        ↓
   股市反应：成长股/科技股对利率最敏感（DCF 估值模型分母变化）
        ↓
   板块轮动：若利率上行 → 资金从成长股流向价值股/防御板块
   ```

   **CME FedWatch 是关键观测工具**：`web_search` "CME FedWatch" 可直接看到市场对下次 FOMC 会议加息/降息的概率定价，数据公布后几分钟内概率会跳变。

   #### 5.3 核心数据场景矩阵

   **CPI（通胀数据）**：

   | 场景 | 市场解读 | SPY/QQQ 预期反应 |
   |------|---------|-----------------|
   | CPI 超预期（高通胀） | 降息推迟/加息预期升温 → 利率上行 → 利空 | ↓ 下跌，科技股领跌 |
   | CPI 符合预期 | 已定价，短期中性 | → 震荡 |
   | CPI 不及预期（通胀降温） | 降息预期升温 → 利率下行 → 利多 | ↑ 上涨，成长股领涨 |

   **非农就业（NFP）**：

   | 场景 | 市场解读 | SPY/QQQ 预期反应 |
   |------|---------|-----------------|
   | 就业超预期（强就业） | **双重解读**：若通胀担忧主导 → 利空；若软着陆叙事主导 → 利多 | 需结合同时公布的失业率和时薪判断 |
   | 就业大幅不及预期（< 10 万） | 衰退担忧 → 避险情绪 → 利空 | ↓ 下跌 |
   | 就业温和增长（10-20 万） | 软着陆叙事 → 经济不热不冷 → 利多 | ↑ 温和上涨 |

   **关键注意**：非农往往与**失业率 + 平均时薪**同时公布，三者必须联合解读。例如：
   - 就业超预期 + 时薪增速低 → 软着陆（利多）
   - 就业超预期 + 时薪增速高 → 通胀担忧（利空）

   #### 5.4 当前宏观周期定位

   | 宏观组合 | 市场阶段 | 数据关注重点 |
   |---------|---------|------------|
   | 经济强 + 通胀下降 | **软着陆（Goldilocks）** — 最利好 | CPI 确认通胀下行趋势 |
   | 经济强 + 通胀高企 | **过热** — 警惕加息 | CPI + PPI + 时薪增速 |
   | 经济弱 + 通胀下降 | **降温** — 期待降息 | 非农 + 失业率确认衰退程度 |
   | 经济弱 + 通胀高企 | **滞胀** — 最差组合 | 所有数据，市场无处可躲 |

   #### 5.5 数据日历与提前量窗口

   AI 必须根据分析周期，搜索对应窗口内即将公布的关键数据。**窗口外的事件无需过度关注——不到提前量，就是噪音。**

   | 分析周期 | 提前量窗口 | 搜索指令 |
   |---------|-----------|---------|
   | 盘中周期 | 当天 | `web_search` "economic calendar today" |
   | 短线周期 | 未来 10 天 | `web_search` "economic calendar this week next week" |
   | 中期周期 | 未来 3 个月 | `web_search` "economic calendar {当前月份} {当前年份}" |

   搜索后必须列出窗口内的**具体日期和数据名称**，例如：
   - 6 月 13 日（明天）：CPI 公布 → 参考 5.3 场景矩阵预判
   - 6 月 14 日（后天）：FOMC 利率决议 + 点阵图 → 可能变盘点

   **关键数据优先级**：

   | 优先级 | 数据 | 公布频率 |
   |--------|------|---------|
   | ⭐⭐⭐ 最高 | 非农就业 + 失业率 + 平均时薪 | 每月第一周周五 |
   | ⭐⭐⭐ 最高 | CPI（通胀核心） | 每月中旬 |
   | ⭐⭐⭐ 最高 | FOMC 利率决议 + 点阵图 + 鲍威尔讲话 | 每年 8 次 |
   | ⭐⭐ 高 | PPI、零售销售、PCE | 每月 |
   | ⭐ 中 | GDP、PMI、消费者信心 | 每月/每季度 |

   > ⚠️ 数据公布前后市场常出现「预期交易 → 公布瞬间跳空 → 消化后反向」的三段式走势。

   **输出规则**：
   - 若窗口内有数据 → 列出具体日期和名称，结合 5.3 场景矩阵预判可能影响方向
   - 若窗口内无关键数据 → 写「当前周期内无关键经济数据公布，数据风险低」
   - **严禁在没有数据的日子硬编数据风险**

**宏观结论输出**：大盘方向（看多 / 看空 / 震荡）+ 关键依据

### Step 1.5：资金面与现金流分析

宏观流动性决定「池子有多大」，但市场内部资金面决定「水够不够」。以下三个维度用于判断市场增量资金状况和潜在抽血风险：

#### 1. 整体持仓杠杆率（Margin Debt）—— 杠杆推上去的钱，跌的时候会加倍抽走

| 指标 | 获取方式 | 三层判断 |
|------|---------|---------|
| FINRA Margin Debt 总量 | `web_search` "FINRA margin debt latest" | **数值**：> $700B 为高杠杆参考区。**趋势**：连续 2 个月下降 → 去杠杆。**市场确认**：Margin Debt 高 + 连续下降 + SPY/QQQ 回落 → 去杠杆风险确认；若指数仍强 → 可能有新资金入场替代杠杆资金 |
| Margin Debt 环比变化 | 同上，对比上月数据 | 同上 |
| Margin Debt / 总市值 | 自行计算 | 辅助参考，需结合趋势判断 |
| 自由信用余额（Free Credit） | FINRA 月报 | 持续下降 + 散户持仓集中股(如 GME/RDDT)同步回落 → 散户购买力枯竭确认 |

**对现金流的传导链条：**
```text
保证金债务↑ → 增量杠杆资金入市 → 推高资产价格
保证金债务见顶并拐头↓ → 增量资金消失 → 市场失去支撑
保证金债务持续↓ → 强制平仓引发卖盘 → 流动性踩踏抽离 → 下跌加速
```

**实战判断：**
- Margin Debt 处于历史高位 + 拐头向下 = 减仓/风控信号
- Margin Debt 低位回升 = 新一轮加杠杆周期的早期信号（利多）
- 2021 年 10 月 Margin Debt 见顶 $936B → 随后纳指下跌 33%

#### 2. 重大 IPO 募资 —— 大公司上市是抽水机

| 指标 | 获取方式 | 阈值判断 |
|------|---------|---------|
| 当月/当周 IPO 募资总额 | `web_search` "US IPO volume this month" | 单月 > $100 亿 → 显著流动性压力 |
| 大型 IPO 日历（募资 > 10 亿） | `web_search` "largest IPOs upcoming" | 当月 ≥ 3 家大型 IPO → 密集抽血 |
| IPO 首日平均表现 | `web_search` "IPO first day performance" | 破发率高 → 市场承接力不足 |
| 科技/成长股 IPO 占比 | Renaissance IPO ETF 或搜索引擎 | 占比高 = 市场情绪过热信号 |

**对现金流的传导链条：**
```text
大型 IPO 募资 → 机构卖出持仓筹集认购资金 → 二级市场被动抽血
IPO 密集期叠加 → 冻结资金量激增 → 短期流动性紧张（通常 1-2 周）
IPO 上市后表现差 → 打新资金亏损 → 后续参与意愿下降 → 一级半市场冷却
锁定期解禁（90-180 天）→ 大量解禁股涌入 → 个股及板块承压
```

**实战判断：**
- 大型 IPO 扎堆 = 市场顶部信号（公司抓紧窗口变现）
- IPO 频繁破发 = 市场资金已不足以支撑新供给 → 警惕
- 注意上市后 90-180 天的解禁窗口对个股的冲击

#### 3. 增发融资（Secondary / Follow-on Offering） —— 现有公司抽血更直接

| 指标 | 获取方式 | 阈值判断 |
|------|---------|---------|
| 月度增发总额 | `web_search` "US secondary offerings monthly total" | > $200 亿/月 → 高抽血状态 |
| 增发 / IPO 比值 | 自行计算 | 比值飙升 → 企业急于高位变现 |
| 科技成长股增发数量 | `web_search` "secondary offering tech stocks" | AI/半导体频繁增发 = 板块估值见顶 |
| ATM (At-The-Market) 发行公告 | 财报/公司公告 | 持续 ATM → 压制股价的上方供给 |

**对现金流的传导链条：**
```text
增发公告发布 → 股价短期承压（稀释预期 + 供给增加）
大规模增发潮 → 大量资金被锁在一级半市场 → 二级市场缺血
行业集中增发（如 AI / 新能源）→ 板块资金面整体恶化 → 板块见顶
增发资金用途 → 还债 = 利空 / 扩产 = 中性 / 并购 = 看并购标的质地
```

**实战判断：**
- 增发是比 IPO 更直接的抽血方式（成熟公司量大且频繁）
- 大科技/成长股集中增发期间该板块难有超额收益
- 2020-2021 增发潮 = 大量公司高位套现 → 随后的 2022 大跌
- 关注增发用途：还债型增发是明显利空信号

**资金面综合结论输出**：现金流充裕 / 中性 / 偏紧 / 紧张 + 关键依据

### Step 2：板块轮动分析

获取各板块核心标的的最新表现：

| 板块 | 观察标的 | 获取方式 | 判断 |
|------|---------|---------|------|
| AI 链 | NVDA, MSFT, META, GOOG, AMZN | `yahoo-finance` | NVDA 强 → AI 强 → 风险偏好高 |
| 半导体 | SOXX, AMD, AVGO, TSM, MU | `yahoo-finance` | 科技牛市发动机 |
| 金融 | XLF, JPM, GS, BAC, MS | `yahoo-finance` | 经济健康程度 |
| 能源 | XLE, 原油 | `yahoo-finance` / `web_search` | 通胀预期 |
| 防御板块 | XLV(医疗), XLP(消费), XLU(公用) | `yahoo-finance` | 避险情绪 |

**板块轮动口诀**（仅作为阶段感参考，不作单独判断依据）：

```text
AI领涨 → 半导体跟涨 → 软件跟涨 → 小盘股补涨 → 周期股补涨 → 防御板块接棒 → 熊市
```

> ⚠️ 该口诀仅为历史规律的简化描述，不可机械套用。实际判断必须结合板块相对强弱、指数趋势、市场广度和龙头股确认。

**板块结论输出**：当前主线板块 + 轮动阶段 + 依据

### Step 3：风格轮动分析

| 观察对象 | 获取方式 | 判断 |
|---------|---------|------|
| SPY, QQQ, IWM | `yahoo-finance` | QQQ/SPY 比值判断成长风格 |
| VUG, VTV | `yahoo-finance` | VUG 强 = 牛市风格 / VTV 强 = 防御风格 |

**风格结论输出**：大盘/小盘 + 成长/价值

### Step 3.5：价格结构与市场广度确认

宏观和板块分析告诉你「应该发生什么」，价格结构告诉你「市场是否真的在这样走」。本 Step 必须执行，不得跳过。

#### 观察指标与判断

| 指标 | 获取方式 | 判断 |
|------|---------|------|
| SPY vs 关键均线 | `yahoo-finance` | SPY 是否位于 20D / 50D / 200D 均线上方？均线排列方向？ |
| QQQ vs SPY 相对强弱 | `yahoo-finance` | QQQ 强于 SPY → 成长/科技驱动；SPY 强于 QQQ → 价值/防御主导 |
| IWM 是否跟随 | `yahoo-finance` | SPY 涨但 IWM 不跟 → 上涨不健康，缺乏广度 |
| RSP vs SPY | `yahoo-finance` | RSP（等权 SPY）跑输 SPY → 指数由少数大票推动，内部分化 |
| 龙头股位置 | `yahoo-finance` | NVDA/MSFT/AAPL 等是否在创新高？龙头乏力 → 行情进入末期 |
| A/D Line | `web_search` | 指数涨但 A/D 降 → 背离，假突破风险 |
| 新高 vs 新低数量 | `web_search` | 新高数 > 新低数确认趋势；新高数萎缩 → 动能衰减 |

#### 综合判断规则

```text
指数上涨 + 广度改善（IWM 跟涨、RSP 不跑输、A/D Line 向上）  →  趋势健康，看多可加置信度
指数上涨 + 广度恶化（少数大票拉指数、IWM 弱、A/D 背离）    →  结构性分化，警惕假突破
QQQ 强 + IWM 弱                                        →  大盘成长占优，风险偏好未充分扩散
SPY 强 + RSP 弱                                        →  指数由权重股推动，市场内部分化严重
QQQ 弱 + IWM 强                                        →  资金转向小盘/价值，风格切换信号
SPY 跌破 50D + A/D 加速下行                             →  中期趋势转弱，风险升级
```

**价格结构结论输出**：趋势健康 / 结构性分化 / 趋势转弱 + 关键依据

### Step 4：市场情绪与风险分析

**判断原则**：情绪指标不单独使用，必须结合指数走势验证。情绪极端 + 指数确认 = 有效信号。情绪极端但指数反向 = 噪音。

| 指标 | 获取方式 | 三层判断 |
|------|---------|---------|
| VIX | `web_search` "VIX today" | **数值**：< 15 乐观 / 15-25 正常 / 25-30 警惕 / > 30 恐慌。**趋势**：VIX 快速抬升 vs 缓慢上行含义完全不同。**市场确认**：VIX > 25 且 SPY 同步下跌 → 恐慌确认；VIX > 25 但 SPY 横盘 → 可能只是期权因素，不必恐慌 |
| Put/Call Ratio | `web_search` "put call ratio" | 高=看空 / 低=看多，需配合 SPY 走势确认 |
| Fear & Greed | `web_search` "fear greed index" | 辅助参考，不单独使用。极度恐惧 + 指数企稳 = 可能见底；极度贪婪 + 指数滞涨 = 可能见顶 |
| 涨跌家数 / A/D Line | `web_search` | 指数涨但 A/D 降 = 假突破（背离确认），需连续观察 3 日以上 |

**风险结论输出**：主要风险点 + 风险等级

### `核心冲突` 字段说明

市场很少是单一方向的。**真正有价值的分析，不是把所有信号强行统一成单一结论，而是诚实地列出矛盾的信号，并判断哪一方更重要。**

**正反信号必须同时列出**，格式如下：

```json
"核心冲突": [
    "利多：10Y 从 4.6% 回落至 4.3%，利率压力缓解",
    "利多：NVDA 突破新高，AI 链资金持续流入",
    "利空：SPY 创新高但 RSP（等权指数）跑输 2.3%，上涨由少数大票驱动",
    "利空：A/D Line 连续 5 日下降，市场广度恶化",
    "判断：价格结构偏强但广度不足，短期震荡偏多，中期需警惕假突破"
]
```

**填写规则：**
1. 原则上应同时列出利多和利空信号
2. 如果未发现有效反向信号，必须写：「当前未发现足够强的反向信号」，**不得为了格式强行编造**
3. 若缺少反向信号是因为数据缺失，则置信度下调；若数据完整且多维度趋势一致（如 Fed 鸽 + 10Y 下行 + QQQ 创新高 + 广度改善），则不必强行下调置信度
4. 最后一条必须是「判断」—— 说明哪个力量在当前周期更重要
